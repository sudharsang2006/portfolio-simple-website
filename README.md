# portfolio-simple-website
My First Small Project
This is my personal portfolio website built using HTML and CSS, featuring a clean and simple two-page design.

The website consists of a Home page that introduces me and highlights my work, and a Contact page that allows visitors to get in touch. The project focuses on fundamental front-end development concepts, including layout design, styling, and basic navigation.

🚀 Features

Simple and clean two-page structure

Home page with introduction and overview

Contact page for communication

Easy navigation between pages

Responsive and lightweight design

🛠️ Tech Stack

HTML5

CSS3

📌 Purpose

This project was created to practice and demonstrate my understanding of basic web development using HTML and CSS, while building a personal online presence.
